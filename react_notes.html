Components :


Portion of page is called a component.

Used to re-use them as and when required.


U can use or create any button or image or feature as a component. Which you can effectively use as a component to other projects as well.


Props : when you want to send data to a component from external that is called prop. To do re-usability. 

 

State :  Internal to code,



Prop and state both are data only.



Class based component  : state is inbuilt creation here.


Function based component: use hooks to maintain state. Hooks are functions.
